Steps to run the application

1. There are two apps, backend & frontend
2. node v10+
3. Backend (Express) - http://localhost:5000
    a) npm install
    b) npm start
    c) npm test (for test cases)
4. Frontend (React/Redux) - http://localhost:3000
    a) npm install
    b) npm start
    c) npm test (for test cases)
5. For more info refer to README.md in respective apps
6. Possible improvements
    a)  Infinite sequencer may be limit to count n.
    b)  More intractive UI/UX.
    c}  Database integration for better authentication & authorization
